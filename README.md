# array.chunk
[![Build Status](https://travis-ci.org/zhiyelee/array.chunk.svg)](https://travis-ci.org/zhiyelee/array.chunk)

Split array to chunks of given size

## Usage

```js
const chunks = require('array.chunk');

// [[1, 2], [3, 4], [5]]
console.log(chunks([1, 2, 3, 4, 5], 2));
```
